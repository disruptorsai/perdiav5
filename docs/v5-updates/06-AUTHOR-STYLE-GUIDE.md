## Common Article Structure

All GetEducated articles follow a consistent structure:

### 1. Introduction
- Hook related to career goals
- Brief overview of what the article covers
- "GetEducated's Picks" callout (3 featured programs)

### 2. Article Navigation
- Anchor-linked table of contents
- Format: `**ARTICLE NAVIGATION:** [Section 1](#) | [Section 2](#) | ...`

### 3. Attribution Block
```
Written by [Author Name](link)
Expert Review by [Reviewer](link)
Edited by [GetEducated Editorial Team](link)
Updated on [Date]
```

### 4. Main Content Sections
- H2 headings for major topics
- H3 for subtopics
- "GetEducated's Picks" boxes within sections
- Program listings with consistent format:
  - School name (linked to GE school page)
  - Degree name
  - Tuition (In-State & Out-of-State)
  - Accreditation
  - Description

### 5. FAQ Section (when applicable)
- Minimum 3 FAQs
- Uses FAQPage schema

### 6. Conclusion/CTA
- Summary of key points
- Call to action to explore degrees

### 7. Research Attribution
```
### How we researched and created this article:
Last updated on: [Date]
Originally published on: [Date]
Written By: [Author]
Expert Review By: [Reviewer]
Edited By: [Editor]
```

### 8. Sources (when applicable)
- List of external sources cited
- BLS links for salary/job data
- School program pages (internal links only)

## Writing Guidelines

### Tone
- Professional but accessible
- Empathetic to reader's education goals
- Informative without being dry
- Encouraging but not salesy

### Cost Data
- ALWAYS source from GetEducated ranking reports
- Include total cost including fees
- Show both in-state and out-of-state when available
- Never pull from competitor sites

### Linking
- Internal links to GetEducated school/degree pages
- External links ONLY to BLS, government, nonprofit education orgs
- NEVER link to .edu school websites directly
- NEVER link to competitor sites

### Program Listings
- Emphasize "Sponsored Listing" programs (show logo)
- Include non-sponsored for balance
- Always link to GetEducated school page
- Include accreditation info

## AI Prompt Integration

When generating content, the AI should:

1. **Match author to topic:**
   - Teaching/education → Charity (Julia Tell)
   - Rankings/cost analysis → Tony (Kif)
   - Healthcare/professional licensure → Kayleigh (Alicia)
   - General guides/overviews → Sara (Daniel)

2. **Apply author style:**
   - Use the `writing_style_profile` from the database
   - Match tone, complexity, and sentence structure

3. **Include required elements:**
   - GetEducated's Picks boxes
   - Article navigation
   - Proper attribution
   - Update date

4. **Validate before publishing:**
   - Author is one of the approved 4
   - All links comply with rules
   - Cost data from ranking reports
   - Shortcodes used (not raw links)
